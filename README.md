# Expert-Mur-Contreforts
Expertise Mur à Contreforts (BAEL)  Outil d'optimisation complète selon les règles BAEL :  Stabilité : Vérification des pressions sol et basculement.  Ferraillage : Calcul du rideau par tranches et alertes de densité.  Métré : Estimation instantanée Béton/Acier au ml.  Objectif : Réduire les coûts par une optimisation précise de l'entraxe. 🏗️
